Iglyfrog

Whats going on?

When you click the squirrel you are performing a POST to /api/hello-world. This is backed by the Jersey resource HelloWorldResource, which creates a Saying object and asks SayingDao to save it. The DAO then using Hibernate to save the object. Once saved the Saying domain object is returned and displayed.

You can see all the "hello"s said by doing a GET to /api/hello-world.
