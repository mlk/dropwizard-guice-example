Whats going on?

When you click the send button (or press enter) you are performing a websocket PUSH to /chat. This is backed by the resource HelloWorldChatResource, which implements an Atmosphere handler that decodes the JSON representation to a Java class and re-broadcasts it to all listeners as JSON.
